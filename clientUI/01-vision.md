# 01 - Vision

## Norte

A area do cliente deve parecer uma landing page viva do proprio projeto.

Em vez de abrir num sistema com caixas pequenas e labels frios, o cliente deve ver uma pagina com impacto:

- titulo grande;
- status claro;
- proxima acao impossivel de ignorar;
- progresso visual;
- secoes com storytelling;
- botoes grandes e decisivos;
- cards que parecem highlights de projeto, nao planilhas disfarçadas.

## Sensacao desejada

O cliente deve sentir:

- "meu projeto esta sendo cuidado";
- "eu sei exatamente o que fazer";
- "isso parece premium";
- "nao preciso procurar informacao";
- "a equipe esta me guiando".

## O que remover do clima atual

- excesso de cards pequenos;
- interface muito administrativa;
- muito texto em caixa sem hierarquia;
- navegacao lateral com cara de sistema;
- telas que parecem vazias quando ha pouco dado;
- headings pequenos demais;
- CTAs secundarios competindo com a acao principal.

## Nova linguagem visual

Direcao: editorial, premium, clara.

Base:

- fundo limpo com bandas largas;
- contraste forte entre hero e conteudo;
- tipografia grande em momentos-chave;
- cards grandes apenas quando ajudam a contar historia;
- secoes full-width com respiro;
- botoes com labels diretos;
- icones como apoio, nao decoracao solta.

Evitar:

- muitos gradientes;
- bolhas decorativas;
- glass exagerado;
- tudo arredondado demais;
- tela cinza sem vida;
- dashboard denso de admin.

## Metafora de produto

O cliente nao esta "usando um painel".

Ele esta acompanhando uma jornada:

1. O projeto agora.
2. O que a MAGUI acabou de fazer.
3. O que falta para avancar.
4. O que o cliente precisa aprovar/enviar.
5. O historico completo quando ele quiser aprofundar.
