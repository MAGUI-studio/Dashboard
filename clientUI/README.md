# Client UI Redesign Plan

Este diretorio define a nova experiencia da area do cliente.

Objetivo: sair de "dashboard frio" e virar uma experiencia tipo landing page privada: narrativa forte, hero grande, CTAs claros, progresso visual, secoes com ritmo e linguagem mais humana.

Arquivos:

- `01-vision.md`: direcao criativa, principios e sentimento desejado.
- `02-information-architecture.md`: nova estrutura de telas e navegacao.
- `03-screen-blueprints.md`: desenho funcional de cada tela.
- `04-component-system.md`: componentes novos/retrabalhados.
- `05-copy-and-ux.md`: tom de texto, CTAs e microcopy.
- `06-implementation-roadmap.md`: plano de execucao em fases.

Regra principal: cliente deve entrar e entender em 5 segundos:

1. Onde o projeto esta.
2. O que mudou desde a ultima visita.
3. O que ele precisa fazer agora.
4. Como ver entregas, aprovar, baixar arquivos e responder briefing.
