# Playbook Comercial e Handoff

Este manual orienta a equipe da MAGUI sobre como operar o ciclo comercial dentro da plataforma.

## 1. Gestão de Leads
Cada novo contato deve ser registrado no CRM. Utilize os estágios do Kanban para mover o lead de "Garimpagem" até "Convertido".

## 2. Emissão de Propostas
Sempre gere propostas estruturadas via plataforma.
- **Dica:** Duplique propostas existentes para agilizar o processo.
- **Notas:** Use o builder de notas para descrever o escopo de forma que o time operacional consiga executar sem dúvidas.

## 3. Contratos e Assinaturas
Após o aceite da proposta, gere o contrato oficial.
- O sistema utiliza o **Autentique** para assinaturas digitais.
- Certifique-se de que os dados das partes (CNPJ, Endereço) estão corretos no perfil de faturamento.

## 4. O Gatilho de Handoff
Ao marcar uma proposta como **ACEITA**, o sistema:
1. Cria o projeto operacional automaticamente.
2. Gera o checklist de kickoff.
3. Notifica o cliente para preencher o briefing.

**Atenção:** Nunca inicie a execução técnica sem o handoff formal concluído no sistema.
