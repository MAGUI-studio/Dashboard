# Changelog da Plataforma

Acompanhe as evoluções técnicas e funcionais da MAGUI.studio.

## [v1.2.0] - 2026-04-24
### Adicionado
- **AI Bot Contextual:** Assistente inteligente para clientes baseado nos dados do projeto.
- **Módulo PWA:** Plataforma agora instalável e com suporte offline via Serwist.
- **Gestão Jurídica:** Criação de contratos estruturados e integração com Autentique.
- **Módulo Financeiro:** Controle de faturas, parcelas e anexos de comprovantes.
- **Briefing Expandido:** Novo sistema de onboarding operacional com coleta de ativos de marca.
- **Centro de Comunicação:** Threads estruturadas por contexto para eliminar WhatsApp.

## [v1.1.0] - 2026-04-17
### Adicionado
- **CRM Avançado:** Kanban e gestão de atividades de leads.
- **Timeline do Projeto:** Registro automático de auditoria e updates.

## [v1.0.0] - 2026-04-10
### Adicionado
- **Lançamento Inicial:** Portal do Cliente e Dashboard Admin base.
